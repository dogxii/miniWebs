## 自定义视频播放器

使用 HTML5 视频元素及其 JavaScript API 的自定义视频播放器，带有自定义设计

- **项目链接**:
  - [GitHub 仓库](https://github.com/dogxii/miniWebs/tree/master/Origin/custom-video-player)
  - [在线演示](https://vanillawebprojects.com/projects/custom-video-player/)

### 项目规格

- 使用 CSS 样式显示自定义视频播放器
- 播放/暂停
- 停止
- 视频进度条
- 设置进度条时间
- 以分钟和秒显示时间
